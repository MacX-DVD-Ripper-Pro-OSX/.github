## How to Install

1. Download the latest version of **MacX DVD Ripper Pro OSX** from the release page.  
2. Open the `.dmg` file and drag the application into your `Applications` folder.  
3. Launch the app. If macOS shows a security prompt, open **System Settings → Privacy & Security** and press **Open Anyway**.  
4. Activate your lifetime license 
5. Insert your DVD and start ripping with a preferred output preset.
